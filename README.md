# FOSSEE_Project — Binary Tree Implementation + YAML Integration
By- Raksha Vindya A

This repository provides a **Binary Tree** implementation in Python with **YAML** import/export support, along with a demo and a test script.

## Project files

- `tree.py` — Binary Tree / Node implementation + YAML integration helpers.
- `sample_tree.yaml` — Example YAML file to load a tree.
- `demo.py` — Example usage (build/load and print the tree).
- `test_tree.py` — Basic tests for tree functionality.
- `requirements.txt` — Python dependencies (includes PyYAML).

## Requirements

- Python 3.x
- Install dependencies from `requirements.txt`.
